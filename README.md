## 升级Astro

```sh
npm install astro@lastest
```

## 升级Tailwindcss

```sh
npm install -D tailwindcss@latest postcss@latest autoprefixer@latest

npx tailwindcss -i ./src/tailwind.css -o ./src/tailwind.css --watch
```

## 升级daisyui

```sh
npm i -D daisyui@latest
```
